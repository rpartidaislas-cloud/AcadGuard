# 🛡️ AcadGuard

**Plataforma de Evaluación Académica Anti-IA**

Sistema de aplicación de exámenes con vigilancia activa contra el uso de inteligencia artificial y herramientas externas no autorizadas.

## Contenido del repositorio

- `sistema_examen.html` — Plataforma principal de exámenes (admin + alumno)
- `practica_excel.html` — Hoja de cálculo de práctica con 3 ejercicios
- `Guia_Examen_Financiamiento_Empresarial.docx` — Guía de estudio
- `Guia_Examen_Administracion_Financiera.docx` — Guía de estudio

## Uso

Abre los archivos `.html` directamente en cualquier navegador (Chrome, Firefox, Edge). No requieren instalación ni servidor.

**Credenciales admin por defecto:** `2960168 / 2960168`

## Características

- Vigilancia anti-trampa: detección de cambio de pestaña, pérdida de foco, salida de pantalla completa, atajos prohibidos
- 8 juegos bonus configurables por sección (Pong, Memoria, Caza-Topos, Guitar Hero, Rompecabezas, Trivia, Crucigrama, Sopa de Letras)
- Hoja de cálculo integrada con motor de fórmulas
- Horarios programables por examen y por alumno individual
- Analítica con gráficos SVG e interpretaciones automáticas
- Cuestionario confidencial de retroalimentación post-examen
