Antes de iniciar tu examen

⏱️ Tiempo

Tendrás 120 minutos para resolver todo el examen. El temporizador inicia al hacer click en "Comenzar examen".

Cuando se acabe el tiempo, el examen se entregará automáticamente con tus respuestas actuales.

🚫 Reglas anti-trampa

El sistema detecta automáticamente:

  • Cambio de pestaña o ventana (Alt+Tab, minimizar)
  • Pérdida de foco del navegador
  • Salida del modo pantalla completa
  • Atajos de copiar, pegar, imprimir o abrir DevTools
  • Inactividad por más de 10 segundos

Tienes 3 alertas como máximo. Después de la 3ª alerta, el examen se enviará automáticamente.

Cuando aparezca una advertencia, el cronómetro NO se detiene. Tendrás 5 segundos antes de poder cerrarla. Al terminar la cuenta regresiva se te asignará 1 alerta(s).

🖥️ Pantalla completa obligatoria

Al hacer click en "Comenzar examen" se activará pantalla completa automáticamente. Si presionas ESC o sales del modo pantalla completa, se levantará una advertencia y volveremos a entrar.

🚪 Salida del examen

Si cierras la pestaña o el navegador durante el examen, se te asignarán 5 alertas como penalización.

Si tienes un problema técnico, tendrás 2 minuto(s) para reingresar con tu código sin perder tu progreso. El cronómetro NO se detiene durante este tiempo.

Pasado ese tiempo, deberás contactar a tu docente con evidencia para que te re-autorice el examen.

🚀 Configuración rápida (5 minutos):
  1. Ve a supabase.com y crea una cuenta gratis

  4. Ve a Project Settings → API y copia la URL y el Anon Key

📖 Formato de importación AcadGuard

El formato AcadGuard-v1 es un JSON legible y editable diseñado para crear exámenes rápidamente. Soporta tres tipos de pregunta:

  • opcion_multiple — Pregunta con 2 a 6 opciones, una correcta
  • abierta — Pregunta de texto libre que el docente revisa manualmente
  • relacion_columnas — Relacionar concepto con su definición

Estructura básica

{
  "formato": "AcadGuard-v1",
  "titulo": "Examen Final Financiamiento Empresarial",
  "minutos": 120,
  "horarioInicio": null,
  "horarioFin": null,
  "secciones": [
    {
      "id": "I",
      "nombre": "Conceptos básicos",
      "peso": 30,
      "preguntas": [ ... ]
    }
  ]
}

Pregunta de opción múltiple

{
  "tipo": "opcion_multiple",
  "puntos": 5,
  "enunciado": "¿Qué es la liquidez de una empresa?",
  "opciones": [
    "Capacidad de obtener ganancias",
    "Capacidad para cumplir obligaciones de corto plazo",
    "Capacidad de pago a largo plazo",
    "Cantidad de efectivo disponible"
  ],
  "respuesta_correcta": 1
}

Nota: respuesta_correcta es el índice (empezando en 0). En el ejemplo, la opción correcta es la segunda (índice 1).

Pregunta abierta

{
  "tipo": "abierta",
  "puntos": 10,
  "enunciado": "Explica con tus palabras qué es el WACC.",
  "criterios_evaluacion": "Mencionar costo promedio ponderado, ponderación E/V y D/V, escudo fiscal."
}

Relación de columnas

{
  "tipo": "relacion_columnas",
  "puntos": 8,
  "enunciado": "Relaciona cada concepto con su definición:",
  "pares": [
    { "concepto": "CETE", "definicion": "Bono cupón cero del Gobierno Federal" },
    { "concepto": "BONDE", "definicion": "Bono con tasa revisable" },
    { "concepto": "UDIBONO", "definicion": "Bono indexado a la UDI" }
  ]
}
💡 Tip: Pídele a tu autor de contenido el examen ya en este formato. Lo subes con el botón "📥 Subir archivo .json" y queda listo en segundos. Para descargar un examen existente en este formato, usa el botón 📤 Exportar en cada examen.
